Getting Started with Angular

A quick guide to help you set up your Angular development environment.

Congratulations on starting your Angular journey! Angular is a powerful platform for building web applications. This application demonstrates the core features of Angular with a responsive design.

Angular Setup Guide

1. Install Prerequisites

Before creating an Angular project, you need to install:

  • NVM for Windows https://github.com/coreybutler/nvm-windows
  • Node.js (version 18 or later) - Includes npm or better
  • npm (Node Package Manager) or yarn
  • TypeScript: Install with npm install typescript --save-dev. More info at typescriptlang.org https://www.typescriptlang.org/download/

    • 2. Install Angular CLI

    Open your terminal and run:

    npm install -g @angular/cli

    Open your terminal and run:

    ng --version

    3. Create a New Project

    Generate your Angular application with:

    ng new my-angular-app

    4. Run the Application

    Navigate to your project folder and start the server:

    cd my-angular-app
    ng serve --open

    Your app will automatically open in your default browser at

    http://localhost:4200/
