Rule 2: Don’t Use the ELSE Keyword

Prefer early returns to simplify logic

Em Angular, componentes limpos evitam `else`. Ao retornar antecipadamente, a lógica fica mais direta e testável.

Exemplo no Angular

Evite:

            
if (this.hasPermission) {
  this.loadData();
} else {
  this.redirect();
}

Prefira:

            
if (!this.hasPermission) return this.redirect();
this.loadData();
