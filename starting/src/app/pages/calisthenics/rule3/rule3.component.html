Rule 3: Wrap All Primitives and Strings

Avoid primitive obsession using strong types

No Angular, evite passar strings ou números diretamente entre serviços/componentes. Use interfaces ou classes para encapsular dados.

Exemplo no Angular

Evite:

            
login(user: string, pass: string) {...}

Prefira:

            
interface Credentials {
  username: string;
  password: string;
}
login(credentials: Credentials) {...}
