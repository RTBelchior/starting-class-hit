Rule 5: One Dot per Line

Avoid chaining too many calls on a single line

Correntes de chamadas dificultam testes e depuração em Angular. Uma linha = uma chamada.

Exemplo no Angular

Evite:

            
this.authService.getUser().getCompany().getName()

Prefira:

            
const user = this.authService.getUser();
const company = user.getCompany();
const name = company.getName();
